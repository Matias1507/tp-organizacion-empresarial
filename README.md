### Escenario Seleccionado:
**Escenario B - Análisis de Ventas de una Pequeña Empresa**
Se desarrolló un script automatizado en Python para procesar la información comercial de la empresa, calcular las métricas principales de desempeño y exportar los resultados gráficos.

---

### Estructura del Repositorio:
* `/datos`: Contiene el archivo `ventas.csv` con los registros diarios.
* `/scripts`: Contiene el código fuente `analisis_ventas.py`.
* `/resultados`: Espacio destinado a los reportes estadísticos y gráficos generados.

---

### Instrucciones de Ejecución:
1. Asegurarse de contar con las librerías `pandas` y `matplotlib` instaladas.
2. Ejecutar el script desde el entorno de Google Colab o entorno local.
3. El programa leerá el dataset de forma relativa y exportará el gráfico automáticamente.
